Welcome to GitHub repository services

This resource provides the source code and components that implement the demonstration Pharmacy Search and results mapping system on VerifyXML.org

http://www.verifyxml.org

Included in the README files and the instructions for configuration and setup of the demonstration system.

Online sample webservices

At the VerifyXML.org website you can access and run live RESTful webservices that show the actual XML exchanges implemented in the demonstration system.

Tutorial Videos

A selection of YouTube videos are available that explain the concepts and approach and then show a hands-on view of creating a sample service from the SQL data stores to the working XML using Open-XDX.
